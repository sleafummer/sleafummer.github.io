1 ） 使用 create-react-app 创建一个 react 项目

由于本地新建一个零配置的 React 单页应用对我来说相对复杂 ， 所以我选择使用 create-react-app 来创建应用 。 事实上 ， 这也是最简单的 。

在这之前你得安装 nodejs

进入到 my-learn-react 目录下 ， 运行 ， 先跑起来 。

我跑起来了 ， 如果你没跑起来 ， 可以加我微信

2 ） 需要关心那些文件

目前 ， 我只需要关心 App.js 和 index.js 这两个文件

同时 ， 我删除了

  1. App.css
  2. App.test.js
  3. index.css
  4. logo.svg
  5. serviceWorker.js

同时 ， 我也删除了 App.js 和 index.js 中对他们的引用 。

最后的文件和内容长这样儿 。

能跑起来就对了 🏃🏻

3 ） 主要文件介绍

index.html -
index.js - 这个 React 项目的入口
App.js - App 组件

目前我们的项目非常简单 ， 只有一个组件儿

不过之后可能会把组件分离到多个文件中 ， 其中每个文件维护一个或者多个特定的组件 。
